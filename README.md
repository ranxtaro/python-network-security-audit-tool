# Python-Based Network Security Audit Tool

Desktop application for automated local network security auditing, developed as a final diploma project.

## Overview

This project is a Python-based tool designed to automate basic security auditing of a local computer network. The application detects active network hosts, scans open ports, identifies network services, analyses potential security risks, and generates structured audit reports.

## Features

- Active host discovery in a local network
- Open port scanning
- Basic network service identification
- Risk analysis based on detected services and ports
- Report generation in structured formats
- Graphical user interface for easier interaction
- Executable build support using PyInstaller

## Technologies Used

- Python
- TCP/IP networking concepts
- Tkinter
- OpenPyXL
- PyInstaller
- PyCharm
